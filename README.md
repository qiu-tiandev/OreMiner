# OreMiner

**Mod ID:** `ore-miner`  
**Latest Version:** v1.1.0  
**Minecraft:** 1.21.10  
**Fabric Loader:** 0.17.3+  

---

## Description

Ore Miner is a Fabric mod that automatically all adjacent ores automatically when a player mines one ore.  
Mod has a toggle command allowing each player to enable or disable OreMiner.

---

## Installation

1. Install **Fabric Loader 0.17.3** for Minecraft 1.21.10  
2. Place `ore-miner-v.x.x.jar` in your `.mods` folder  
3. Start Minecraft with the Fabric profile
5. **Make sure you have the fabric api mod**

---
### Commands
#### `/oreminer`
Toggle Ore Miner on or off for your player account: 

---

## Building from source

1. Clone this repo with `git clone https://github.com/qiu-tiandev/OreMiner`
### Windows
2. Go to the repo directory and run `gradlew.bat build`
3. Check `build/libs` for compiled jar file
### MacOS/Linux
2. Go to repo directory and run `./gradlew build`
3. Check `build/libs` for compiled jar file
